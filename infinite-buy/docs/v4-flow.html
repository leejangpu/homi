3 모드 상태 머신 (V4 핵심)

🆕 새 사이클
T=0 · 보유 0 · 원금=예수금/복리
큰수매수 = 종가 × 1.10 에 LOC
↓ 매수 체결
NORMAL (일반)
• 전반전 (T < 분할/2): 별지점 + 평단 LOC
• 후반전 (T ≥ 분할/2): 별지점 LOC
• + 대폭락 5단 티어
• 매도: 1/4 별지점 LOC + 3/4 목표가 지정가
↓
보유 0
🏁 사이클 종료
실현손익 확정 → 복리로 새 사이클 재시작 (위로)
T > 분할수−1 (1회분 미만)
REVERSE (소진 후)
• 기준선 = 직전 5일 종가평균
• 매도: 1일차 MOC → 이후 LOC 등분
• 매수: (잔금+매도금)/4 쿼터
• T: 매도 ×(1−2/n) · 매수 +(n−T)×0.25
↑ 종가 > 평단 ×(1−목표%) 회복 → NORMAL 복귀 (T 그대로 승계)
새 사이클 NORMAL REVERSE 사이클 종료

4 국면별 주문 요약

국면매수매도
새 사이클 (T=0) 큰수 = 종가×1.10 LOC + 하방 5단 티어 —
전반전 (T < 분할/2) 1금 절반 별지점, 절반 평단 LOC + 티어 1/4 별지점 LOC + 3/4 목표가 지정가
후반전 (T ≥ 분할/2) 1금 전액 별지점 LOC + 티어 〃
리버스 1일차 🚫 매수 금지 보유 ÷(분할/2) MOC
리버스 2일차~ (잔금+매도금)/4 → 5일평균 아래 LOC 보유 ÷(분할/2) → 5일평균 위 LOC

별% = 목표 − (2·목표/분할수)·T · 별지점 = 평단×(1+별%) · 매수는 별지점−$0.01 · 1회매수금 = 잔금/(분할수−T)
