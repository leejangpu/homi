> **출처**: `unisheet-v2.2-source.xlsx` (같은 폴더에 저장). 사용자가 언이시트로 만든 v2.2 시트("라오어 무한 매수법 언이시트 v1.2"). **스크립트 없음** — 셀 수식으로만 계산되는 자동계산 시트. `기본1` 시트 기준.

> v4의 언이시트(`unisheet-v4-source.gs.txt`, Apps Script)와는 **다른 파일**이다. v4는 소진 처리를 리버스모드로 하지만, 이 v2.2 시트는 **쿼터손절모드**를 쓴다. v2.2 쿼터손절의 유일한 권위 출처다.

## 시트 구조 (기본1)

- 우측 계산 블록: `BT~CW` 열이 V1/V2/V2.1/V2.2 각각의 매수/매도/큰수/수량을 계산. **V2.2 = `CR~CW`**.
- 좌측 `A~K`(행25~) = 종목별 표시/입력 계산기. 아래쪽 `B48~`, `B63~`가 **쿼터손절모드** 상세.
- `쿼터손절모드` 토글: `G25`(종목1) 등 TRUE/FALSE 수동 체크.
- 주요 셀: `AC7`=평단, `AH7`=현재가, `O7`=1회매수액, `Z7`=잔고수량, `M7`=분할, `J7`=목표수익률, `J19`=T값, `BG16`=큰수기준(0.15).

## 권위 사양 (종목1 = 행51 수식 해독)

### 진입/자금확보 (매도 → MOC 전환)
```
매도 MOC 트리거 (CW6): AND( 39×(M/40) < T,  T <= 40×(M/40) )
  → 1회분 미만 구간이면 매일 1/4 매도를 MOC로 강제
  · 20분할: T > 19.5   · 40분할: T > 39
MOC 매도량 (CU6/BF51): ROUND(보유/4)     ← 반올림. 1~2주면 0 가능
매도 나머지 3/4: 목표가 LIMIT = 평단×(1+목표+수수료)  (CW8)  ← 그대로 유지
```

### 시드/매수 (쿼터손절 매수)
```
시드 (Z51):          남은원금 + 수익금 + 1/4매도금(C)
쿼터 1회매수금 (AC51): min( 시드/10, 원래 1회매수액 O7 )   ← 시드/10, 원1회금 상한
쿼터 매수 별지점 (BB51): 평단 × 0.9 − 0.01   (= 평단 −10%, 리터럴 0.9)
쿼터 매수 큰수 (BD51):   현재가 × 1.15         (= +15%, 리터럴)
쿼터 매수가 = min(별지점, 큰수),  매수량 = floor(1회매수금 / 매수가)
진행률 (AX51): 남은시드 < 현재가(1주도 못 삼) → 100% 완료
```

### 반복(연속) 쿼터손절
- 한 시드를 다 쓰면(진행률 100%) 여전히 소진 상태면 **다시 1/4 MOC → 새 시드 → 또 배치**.
- `B63~B70`에 **연속 쿼터손절 최대 5회** 기록 테이블 (시작일/수량/평단/남은원금/수익금/매도금/시드/…).
- 즉 **MOC는 데일리가 아니라 "시드 배치당 1회"**.

## 핵심 상수 (리터럴 — 목표수익률과 무관)

| 값 | 셀 | 성격 |
|---|---|---|
| 쿼터 매수 별지점 = 평단×**0.9** (−10%) | `BB51` 리터럴 `0.9` | **목표수익률(J7) 무관, 고정.** 이 시트 종목은 J7=10%라 −10%와 우연히 일치 |
| 쿼터 큰수 = 현재가×**1.15** (+15%) | `BD51` 리터럴 `1.15` | 큰수기준(BG16) 설정과 무관, **고정 +15%** |
| 일반 큰수 = 현재가×(1+**BG16**) | `CW5` | 큰수기준 설정값(이 시트 0.15=15%) |
| 일반 별% = 목표 − T×목표×2/M | `CR4` | 목표(J7) 기반 (우리 코드와 동일) |

> ⚠️ **−10% 해석 주의**: `BB51`은 리터럴 `0.9`라 목표수익률을 바꿔도 항상 −10%로 동작한다. 다만 이 시트의 종목이 목표 10%라 "−10% = −목표"와 구별이 안 된다. 목표 20% 종목(예: SOXL) 시트가 없어 "−10% 고정"인지 "−목표"인지 단정 불가. **우리 코드는 −목표**(calculateQuarterModeStarPercent)라 목표10% 종목은 시트와 일치, 목표20%는 −20%로 갈림. → 미결(사용자 판단).

### ⚠️ 알려진 이슈: 쿼터 매수/매도 별지점 역전
쿼터 **매수** 별지점을 평단×0.9(−10%)로 올렸는데, 우리 코드의 쿼터 **매도**는 여전히 평단×(1−목표)=−20%(SOXL) 조건부 LOC라 **매수($180)가 매도($160)보다 위**에 온다. 종가가 그 사이면 같은 날 매수·매도 동시 체결 → churn. **근본 원인**: 언이시트는 쿼터 존에서 1/4 매도를 **MOC**(무조건, 별지점 없음)로 하는데 우리는 조건부 LOC로 구현했기 때문. **해결책**: 쿼터모드 활성 시 1/4 매도를 MOC로 변경(시트 CW6과 일치, 역전 해소). SOXL T=9.45라 쿼터모드까지 여유 있어 급하진 않음.
