# MVP Roadmap

## Phase 1. 프로젝트 부트스트랩
- Next.js + TypeScript 초기화
- Firebase Auth 연결
- 환경 변수 설정

산출물:
- 로그인 가능한 기본 앱

## Phase 2. 인증/공간/권한
- 회원가입/로그인/로그아웃
- 개인 공간 자동 생성
- 공유 공간 생성 및 멤버 초대

산출물:
- 사용자별 접근 제한 동작 확인

## Phase 3. 모듈 프레임워크
- 모듈 생성/목록/삭제
- 모듈 타입별 공통 인터페이스 정의
- 동일 타입 다중 인스턴스 생성 지원

산출물:
- 개인/공유 대시보드에서 모듈 카드 렌더링

## Phase 4. 기본 모듈 3종
- 가계부(수입/지출 CRUD)
- ToDo(할일 CRUD + 완료 처리)
- 일기(작성/수정/조회)

산출물:
- 실제 사용 가능한 3개 기능 제공

## Phase 5. 통계/차트
- 월별 집계 생성
- 대시보드 차트 연결

산출물:
- 최소 1개 지표(예: 월별 지출 합계) 시각화

## Phase 6. 운영 안정화
- 에러 로깅/성능 점검
- 백업/복구 절차 정리
- 권한 테스트 및 회귀 테스트

## 이후 확장 (AI)
- 자연어 명령 -> 안전한 액션 실행 파이프라인 구축
